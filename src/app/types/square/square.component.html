<spinner [type]="'square'" [shape]="'starter'" [diameter]="'72'" [stroke]="'#bada55'" [width]="'5'" [speed]="'1.2'"></spinner>

Generates the following markup:

        <div _ngcontent-dmt-3="" class="sh-spinner sh-spinner-square">
          <svg _ngcontent-dmt-3="" class="sh-spinner-square fadeOut" width="900" height="96" viewBox="0 0 900 96">
              <g _ngcontent-dmt-3="" transform="translate(411, 9)">
                <rect _ngcontent-dmt-3="" fill="none" stroke-miterlimit="10" x="0" y="0" id="starter0" stroke="#bada55" opacity="1" stroke-width="3" stroke-linecap="round" rx="3" ry="3" height="59.575222039230624" width="59.575222039230624" stroke-dasharray="[movable target]">
              </rect>
            </g>
          </svg>
        </div>

Almost all of the elements of the svg rect are available. In this instance however, [diameter] is the width and height of the rect object. All of the other attributes are essentially the same as the 'circle' type

The colours specified in the [stroke] attribute only accepts a hex colour. This is because, using either rgb, rgba, hls or hlsa throws a 'foo is not a function' error.
