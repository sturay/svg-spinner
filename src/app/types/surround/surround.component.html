This started off as a fanciful idea and, to be honest, I haven't really found a decent use for it yet, but this one essentially covers the parent element to prevent editing.

It takes the height and width of the parent item and applies it to an absolutely positioned element. This means that thew element it is in - its direct parent must have a relative position applied.

Also, there are two other options which relate to this type only and they are [overlay] and [alpha]. [overlay] sets the colour of the fill and must be a hex value while the [alpha] value sets the transparency of the fill. The fill is then returned in an rgba() value.

The colours specified in the [stroke] attribute only accepts a hex colour. This is because, using either rgb, rgba, hls or hlsa throws a 'foo is not a function' error.
